# assets/character ディレクトリ

このディレクトリに以下の5枚の画像ファイルを配置してください。

| ファイル名      | 説明     |
|----------------|----------|
| normal.png     | 通常表情 |
| smile.png      | 笑顔     |
| troubled.png   | 困り顔   |
| angry.png      | 怒り     |
| surprised.png  | 驚き     |

## 推奨サイズ

- 縦長画像（例: 200 × 280px）
- PNG 形式（背景透過推奨）

- [いらすとん](https://www.irasuton.com/)

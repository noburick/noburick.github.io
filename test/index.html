アプリ概要

『ねことも！ポモドーロ』は、ポモドーロ・テクニックをやさしく続けられる集中サポートアプリです。作業中は暖炉の音と猫の見守り演出でほっとひと息、休憩時間は猫とのふれあいで気分転換できます。

主な機能

  • 25分集中 + 5分休憩のポモドーロタイマー
  • 猫のリアクション演出でモチベーションをキープ
  • 暖炉の環境音で落ち着く作業空間を演出
  • 集中回数の記録で毎日の達成感を見える化

スクリーンショット

使い方

  1. 集中したいタスクを決めて「スタート」をタップ
  2. 25分間、猫と暖炉の演出といっしょに作業に集中
  3. 休憩時間は猫をタップしてリラックス
  4. 1日の集中回数を確認して習慣化を続ける
